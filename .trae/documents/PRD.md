# 序列比对工具 PRD (Milestone 1)

## 1. 产品概述

专业的生物序列比对分析工具，支持 DNA/RNA 序列的输入、预处理、比对和结果展示。
- 目标用户：生物信息学研究人员、分子生物学实验室人员
- 核心价值：提供高效、直观的序列比对分析体验，支持大规模数据处理

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，直接使用 | 使用全部序列分析功能，导出分析结果 |

### 2.2 功能模块
1. **输入序列模块**：Reference 序列输入、Query 序列输入、多格式支持、实时统计
2. **序列预处理模块**：反向互补、DNA/RNA 转换、序列清洗
3. **分析设置模块**：比对方法选择、参数配置、任务调度
4. **结果展示模块**：比对结果表格、序列对齐视图、统计信息

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | Header | 应用标题、主题切换、帮助入口 |
| 主页面 | Input Sequence | Reference 序列区域、Query 序列区域、序列统计信息 |
| 主页面 | Analysis | 比对方法选择、参数配置、Analyze 按钮 |
| 主页面 | Result | 结果表格、对齐视图、统计面板、导出按钮 |

## 3. 核心流程

用户从输入序列开始，经过预处理、配置参数、执行比对，最终查看和导出结果。

```mermaid
flowchart LR
    A["输入 Reference 序列"] --> B["输入 Query 序列"]
    B --> C["序列预处理"]
    C --> D["配置比对参数"]
    D --> E["执行比对分析"]
    E --> F["查看比对结果"]
    F --> G["导出结果"]
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：科技蓝（#18A058 变体，Naive UI 风格）
- 辅色调：深灰/浅灰背景，根据主题切换
- 按钮风格：圆角、轻微阴影、hover 动效
- 字体：系统无衬线字体，清晰易读
- 布局风格：顶部导航 + 卡片式分区布局
- 图标风格：简洁线性图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主页面 | Header | Logo、标题、主题切换按钮、帮助按钮 |
| 主页面 | Input Sequence | 双栏布局（Reference / Query）、序列统计卡片、拖放区域 |
| 主页面 | Analysis | 参数配置表单、Analyze 主按钮、进度指示器 |
| 主页面 | Result | 数据表格、详情面板、统计卡片、导出按钮 |

### 4.3 响应式设计
- 桌面端优先（> 1024px）：双栏布局
- 平板端（768px - 1024px）：自适应单栏
- 移动端（< 768px）：单列堆叠布局

### 4.4 动画效果
- 页面切换：平滑过渡
- 按钮交互：hover 缩放 + 颜色渐变
- 内容展开：blinds 效果（新增）、wipe 效果（删除）
- 仅内容区域动画，导航元素不参与动画
